Type of CSS

  1. CSS means Cascading Style Sheets is a language for styling the webpage. Cascading Style Sheets is a simple language design language intended to simplify the process of making webpage presentable.

  2. External CSS

    An external css is a method of applying styles to a webpage by linking an external.css file to an HTML document.This keeps the design separate from the content,making the code cleaner and easier to maintain.


  3. Internal CSS

    Internal CSS is a method of applying styles directly within an HTML document using a style tag inside the head section. This keeps the styles in the same file as the content, making it convenient for single-page projects or quick styling adjustments.


  4. Inline CSS

    Inline CSS is a method of applying styles directly to individual HTML elements using the style attribute. It allows you to define unique styles for specific elements without needing a separate CSS file or style tag.
